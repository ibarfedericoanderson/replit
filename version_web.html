Versión Web de QRGB

Información importante

Esta versión de QRGB está optimizada para ejecutarse directamente en un navegador web, lo que la hace perfecta para alojarla en GitHub Pages u otros servicios de hosting estático.

Ver la versión web del QRGB en línea

Scripts Python con librerías de código abierto (Kivy y Tkinter) para generar códigos QRGB
Acceso directo a GitHub, Streamlit, Google Colab y documentación PDF
Repositorio GitHub y documentación en OSFPreprints

¿Por qué usar la versión web?

La versión web es ideal para compartir rápidamente tu generador de códigos QR con otras personas sin necesidad de que instalen nada. Además, al alojarla en GitHub Pages, tendrás un URL permanente que podrás compartir fácilmente.

Guía rápida de uso

Estructura de archivos

  • index.html - Página principal
  • css/styles.css - Estilos de la aplicación
  • js/qrgb.js - Lógica de la aplicación
  • README.md - Documentación

Configuración en GitHub Pages

  1. Crea un repositorio en GitHub
  2. Sube todos los archivos descargados
  3. Ve a Settings > Pages
  4. En Source, selecciona main
  5. Haz clic en Save
  6. Espera unos minutos a que se publique tu sitio
  7. ¡Listo! Tu URL será similar a username.github.io/repository-name

Limitaciones de la versión web

Ten en cuenta que la versión web tiene algunas limitaciones comparada con la versión de escritorio, como la imposibilidad de decodificar códigos QR existentes y algunas restricciones en la personalización avanzada.

Personalización

La versión web de QRGB es altamente personalizable. Puedes modificar los siguientes aspectos:

Apariencia

  • Colores de la interfaz
  • Tipografía
  • Tamaño de los elementos
  • Disposición de los controles

Funcionalidad

  • Configuración predeterminada
  • Opciones adicionales
  • Formato de salida del QR
  • Tamaños y resolución

Idioma

  • Traducción de la interfaz
  • Mensajes de error
  • Textos de ayuda
  • Documentación
