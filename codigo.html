Código Fuente QRGB

Explora y descarga la aplicación QRGB para generar códigos QR con información en capas RGB

¿Qué es QRGB?

QRGB es una aplicación innovadora que permite generar códigos QR que contienen múltiples mensajes o datos utilizando las capas de color (Rojo, Verde y Azul). Esto permite almacenar hasta tres mensajes diferentes en un solo código QR, que pueden ser leídos individualmente utilizando filtros de color o mediante la aplicación QRGB.

Múltiples Capas

Almacena tres mensajes diferentes en las capas R, G y B de un solo código QR.

Personalización Visual

Crea códigos QR visualmente atractivos con combinaciones de colores personalizadas.

Diferentes Versiones

Disponible como aplicación de escritorio con Python y Kivy, o como aplicación web.

Aplicación de Escritorio con Python y Kivy

La aplicación QRGB para escritorio está desarrollada en Python utilizando el framework Kivy para crear una interfaz gráfica multiplataforma. Permite generar y decodificar códigos QR multicapa, ofreciendo una experiencia completa con todas las funcionalidades.

Requisitos

  • Python 3.6+
  • Kivy
  • Pillow (PIL)
  • qrcode
  • opencv-python

Versión Web con JavaScript

La versión web de QRGB está implementada con HTML, CSS y JavaScript para poder funcionar directamente en el navegador. Es ideal para compartir y usar rápidamente sin necesidad de instalar nada, y se puede alojar fácilmente en servicios como GitHub Pages.

Características

  • Funciona en cualquier navegador moderno
  • No requiere instalación
  • Fácil de compartir con un simple enlace
  • Interfaz intuitiva para generar códigos QR

Aplicación Streamlit

La versión Streamlit de QRGB ofrece una interfaz web interactiva desarrollada con Python. Esta implementación simplifica la generación de códigos QR personalizados con una interfaz minimalista y fácil de usar, aunque solo soporta generación de códigos QR simples (no multicapa).

Ventajas

  • Interfaz web interactiva con Python
  • Fácil de usar con controles intuitivos
  • Personalización de colores y parámetros
  • Desplegada en Streamlit Cloud

Instrucciones de Instalación

Requisitos previos

Antes de instalar QRGB, asegúrate de tener Python 3.6 o superior instalado en tu sistema.

Instalación en Windows

  1. Descarga e instala Python para Windows
  2. Abre una terminal (cmd o PowerShell)
  3. Instala las dependencias: 
    pip install kivy pillow qrcode opencv-python
  4. Descarga y descomprime el archivo ZIP de QRGB
  5. Ejecuta el script con Python: 
    python QRGB.py

Instalación en Linux

  1. Instala Python y las dependencias del sistema: 
    sudo apt-get update
sudo apt-get install python3 python3-pip python3-dev
  2. Instala las dependencias de Python: 
    pip3 install kivy pillow qrcode opencv-python
  3. Descarga y descomprime el archivo ZIP de QRGB
  4. Ejecuta el script con Python: 
    python3 QRGB.py
